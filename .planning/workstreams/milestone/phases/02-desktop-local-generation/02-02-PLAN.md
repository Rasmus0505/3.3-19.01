# Objective

完成 Bottle 1.0 前端到后端的完整生成链路：UploadPanel → desktopRuntime.transcribeLocalMedia() → 本地 ASR → localAsr.generateCourse() → lesson 生成。

## Context

根据 02-CONTEXT.md 决策：
- **D-01:** 用户完全不感知 ASR 来源（云端/本地），无技术细节暴露
- **D-04:** 生成阶段文案：transcribing → generating lesson → completed/failed，复用现有
- **D-13:** 本地 ASR 结果写入 lesson_task 表，统一 pipeline
- **D-16:** asrStrategy.js 已有 Bottle 1.0 本地路由，本阶段完成完整闭环

当前状态：
- preload.cjs 已暴露 desktopRuntime.transcribeLocalMedia() 和 localAsr.generateCourse()
- main.mjs 已有 transcribeLocalMedia() 和 generateLocalCourse() 实现
- asrStrategy.js 有 resolveAsrStrategy() 路由逻辑
- local_asr_assets.py 已有 /api/local-asr-assets/download-models 接口

需要完成：
1. UploadPanel.jsx 中对接 Bottle 1.0 路径
2. asrStrategy.js 中确保 Bottle 1.0 路由正确
3. 确保错误处理不暴露技术细节
