    当 yt-dlp 没有返回 thumbnail（如直链无封面图）时，在 `pollDesktopLinkImportTask` 成功路径中，从已下载的本地文件提取封面。

    场景：链接导入成功（status === "succeeded"），但 `payload.thumbnail` 为空。此时需要：
    1. `sourceFile` 带有 `desktopSourcePath`（指向已下载的本地文件路径）
    2. 调用 `materializeDesktopSelectedFile(sourceFile)` 把文件内容读成 Blob
    3. 对 Blob 调用 `extractMediaCoverPreview` 提取封面 data URL
    4. 把提取结果存入 `sourceFile.thumbnail`

    在 `pollDesktopLinkImportTask` 的 `nextStatus === "succeeded"` 分支中，thumbnail 获取后追加 fallback 逻辑：
    ```js
    if (!thumbnailUrl) {
      // 直链无 yt-dlp 封面，从本地文件提取
      try {
        const materializedFile = await materializeDesktopSelectedFile(sourceFile);
        if (materializedFile && isBlobBackedSourceFile(materializedFile)) {
          const coverPreview = await extractMediaCoverPreview(materializedFile, materializedFile.name || "");
          if (coverPreview?.coverDataUrl) {
            try {
              Object.defineProperty(sourceFile, "thumbnail", { value: coverPreview.coverDataUrl, configurable: true, writable: true });
            } catch (_) { void 0; }
          }
        }
      } catch (_fallbackErr) {
        // 封面提取失败不影响主流程，继续
      }
    }
    ```
  </action>
  <acceptance_criteria>

    - fallback 失败不阻断主流程
