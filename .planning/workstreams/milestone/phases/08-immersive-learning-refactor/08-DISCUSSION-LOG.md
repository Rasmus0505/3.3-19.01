## 句子停留规则

| Option | Description | Selected |
|--------|-------------|----------|
| 答对后回放 1 次再自动下一句 | 完成当前句的播放逻辑后，再完整回放当前句 1 次，然后自动进入下一句 | ✓ |
| 立刻自动下一句 | 答对后不做完成回放，立即推进 | |
| 停留当前句 | 答对后停留当前句，等待手动下一句 | |

**User's choice:** 答对后回放 1 次再自动下一句。
**Notes:** 该规则只在单句循环关闭时生效。

| Option | Description | Selected |
|--------|-------------|----------|
| 持续循环直到主动关闭或手动切句 | 一直停在当前句反复播放，直到用户手动关闭循环或切到其他句 | ✓ |
| 每次播完停住等待手动再播 | 保持当前句，但不自动继续下一轮播放 | |
| 只影响重播按钮 | 正常答对后仍然自动进入下一句 | |

**User's choice:** 持续循环直到主动关闭或手动切句。
**Notes:** 用户明确补充“直到主动关闭循环”。

## 操作冲突优先级

| Option | Description | Selected |
|--------|-------------|----------|
| 硬中断并立即切句自动播放 | 停止当前播放 / 暂停 / 循环，立刻切到目标句并自动播放 | ✓ |
| 切句但默认暂停 | 切到目标句后等待手动播放 | |
| 排队等待当前播放结束 | 等当前句自然结束后再切句 | |

**User's choice:** 硬中断并立即切句自动播放。
**Notes:** 该规则同时适用于播放中和暂停中的上一句 / 下一句操作。

| Option | Description | Selected |
|--------|-------------|----------|
| 立即揭示但不断音 | 揭示字母 / 单词立即生效，不暂停、不重启当前音频 | ✓ |
| 先暂停再揭示 | 揭示前先把当前播放停住 | |
| 字母不断音、单词先暂停 | 对不同揭示动作采用不同音频策略 | |

**User's choice:** 立即揭示但不断音。
**Notes:** 用户要求揭示动作不打断当前听句节奏。

| Option | Description | Selected |
|--------|-------------|----------|
| 上一句单次回听，播完回当前句待命 | 中断当前句，播放上一句 1 次，结束后回到当前句但不自动恢复 | ✓ |
| 上一句单次回听，播完自动恢复当前句 | 中断当前句，播放上一句 1 次，结束后自动继续当前句播放 / 循环 | |
| 只有暂停时才能听上一句 | 避免在播放中打断当前句 | |

**User's choice:** 上一句单次回听，播完回当前句待命。
**Notes:** 用户还要求“上一句”区域右侧只保留一个喇叭按钮承载这个动作。

## 控制面与记忆范围

| Option | Description | Selected |
|--------|-------------|----------|
| 全屏与非全屏都显示循环 / 倍速 | 在所有沉浸视图都常驻暴露这组控件 | |
| 只在全屏答题板块显示循环 / 倍速 | 主路径围绕影院全屏设计，非全屏 fallback 不做完整控制面 | ✓ |
| 非全屏只显示倍速 | 循环仍然收在全屏里，倍速额外外露 | |

**User's choice:** 只在全屏答题板块显示循环 / 倍速。
**Notes:** 这是根据用户“我记得没有这个页面”的反馈做的记录性推断：当前产品心智里没有独立的非全屏沉浸主界面，只有自动全屏失败 / 退出后的 fallback。

| Option | Description | Selected |
|--------|-------------|----------|
| 只有单句循环全局记忆，倍速仅本次会话保持 | 循环开关跨进入次数保留，倍速退出沉浸后恢复 `1.00x` | ✓ |
| 单句循环和倍速都全局记忆 | 下次进入课程继续沿用上次倍速 | |
| 二者都只对当前句生效 | 切句就恢复默认 | |

**User's choice:** 只有单句循环全局记忆，倍速仅本次会话保持。
**Notes:** 用户明确补充：倍速退出沉浸学习后回到默认 `1.00x`。

## 全屏与字幕遮挡板稳定性

| Option | Description | Selected |
|--------|-------------|----------|
| 纯显示层变化，不重置学习状态，并持久化上一句 / 遮挡板偏好 | 全屏、上一句显示、遮挡板开关和拖动都不应打断句子状态；相关 UI 偏好需要记忆 | ✓ |
