新增一个参数 `skipSeek?: boolean`，当为 true 时，强制使用 clip 模式（通过 sentence.audio_url）播放，不影响主视频：

在 `playSentence` 函数开头增加判断：
```js
const playSentence = useCallback(
  async (sentence, playbackPlan = null, { skipSeek = false } = {}) => {
    if (!sentence) { return { ok: false, reason: "sentence_missing" }; }
    stopPlayback();

    // Force clip mode for audio-only playback (no video seek)
    if (skipSeek && sentence.audio_url) {
      mode = "clip";
    }

    const normalizedPlaybackPlan = normalizePlaybackPlan(playbackPlan || {});
    // ... rest unchanged
  },
  [...]
);
```

注意：`playSentence` 在 `useSentencePlayback.js` 中定义为一个普通函数（不是 hook），`mode` 来自函数参数（从 line 182 的 `if (mode === "clip")` 可以看出 mode 是可用的）。
