## 能力深度边界

| Option | Description | Selected |
|--------|-------------|----------|
| 公开链接优先 | 支持常见公开视频页链接和公开直链，不做 cookies、登录态、会员/受限内容导入 | ✓ |
| 公开链接 + 受限内容增强 | 增加浏览器 cookies / 登录态导入，提升受限平台成功率 | |
| 接近 Memo 工具化 | 加上更新、源选择、登录态管理等高级能力 | |

**User's choice:** 不做 cookies、账号登录、会员/受限内容导入。  
**Notes:** 复制 Memo 的边界停在公开链接和常见公开视频页，不进入受限内容绕过能力。

---

## 用户可见控制面

| Option | Description | Selected |
|--------|-------------|----------|
| 完全隐藏高级控制 | 用户只看到标题、链接、进度、失败提示，不暴露媒体工具项 | ✓ |
| 默认隐藏，提供高级入口 | 保留一个高级设置/排障入口 | |
| 明显可见的媒体工具面板 | 接近 Memo 的工具化桌面面板 | |

**User's choice:** 完全隐藏高级控制。  
**Notes:** Bottle 继续走低技术感知路线，不复制 Memo 的工具面板风格。

## 元数据利用程度

| Option | Description | Selected |
