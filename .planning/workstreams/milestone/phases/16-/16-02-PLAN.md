# Phase 16 Wave 2: Admin 公告管理页面

## Objective

在管理台新建「公告管理」Tab 页面，运营人员可创建、编辑、删除和置顶公告。

## Scope

- ANNC-01: Admin 创建公告
- ANNC-02: Admin 标记公告类型
- ANNC-03: Admin 置顶公告
- ANNC-04: Admin 删除公告

## Tasks

### 16-02-TASK-01: 创建 AdminAnnouncementsPage 组件

<read_first>
`frontend/src/features/admin-pages/AdminHealthPage.jsx` — admin tab 页面完整参考
`frontend/src/features/admin-pages/AdminSecurityPage.jsx` — admin tab 页面完整参考
`frontend/src/shared/ui/index.js` — 可用 shadcn 组件导出
`frontend/src/shared/api/adminClient.js` — admin API 客户端调用模式
`frontend/src/features/admin-pages/AdminRedeemPage.jsx` — 大型 admin 页面参考
</read_first>

<action>
在 `frontend/src/features/admin-pages/` 目录下新建 `AdminAnnouncementsPage.jsx`：

**页面结构：**
```
页面标题: "公告管理"
左侧面板 (announcement-list): AnnouncementListItem 列表 (ScrollArea 包裹)
右侧面板 (announcement-editor): AnnouncementEditor 表单 或 EmptyState
```

**AnnouncementListItem（卡片样式）：**
- Title: 18px semibold，1 行截断 ellipsis
- Type Badge: 用 Badge 组件，颜色按 UI-SPEC.md 规定
  - changelog: `bg-secondary text-foreground` + 文字"更新日志"
  - banner: `bg-upload-brand/10 text-upload-brand border border-upload-brand/20` + 文字"公告"
  - modal: `bg-destructive/10 text-destructive border border-destructive/20` + 文字"重要公告"
- Status: is_active=true → 绿色圆点 `bg-green-500` + "已启用"；false → 灰色圆点 `bg-muted` + "已禁用"
- Created date: 13px muted color，`YYYY-MM-DD` 格式
- Hover: `bg-secondary` 背景
- 操作按钮（右侧）: "编辑"（text-upload-brand）+ "删除"（text-destructive）
- 删除需 `AlertDialog` 确认，文案："确认删除此公告？删除后无法恢复。"

**AnnouncementEditor 表单：**
- 标题输入: `Input`，placeholder="请输入公告标题"，maxLength=200
- 类型选择: `Select`，选项：
  - `changelog` → "更新日志"
  - `banner` → "公告"
  - `modal` → "重要公告"
- 内容输入: `Textarea`，placeholder="请输入公告内容（支持 Markdown 格式）"
- 启用状态: `Switch` 开关，左侧标签"启用状态"
- 置顶状态: `Switch` 开关，左侧标签"置顶公告"
- 底部按钮: "保存"（`bg-upload-brand` primary）+ "取消"（secondary）
- 新建时底部额外显示"删除"按钮（destructive），仅已存在公告显示

**API 调用：**
- GET `/api/admin/announcements` — 列表（分页 page/page_size）
- POST `/api/admin/announcements` — 创建
- PUT `/api/admin/announcements/{id}` — 更新
- DELETE `/api/admin/announcements/{id}` — 删除

使用 `adminClient.js` 中的 fetch 模式。操作完成后调用 `queryClient.invalidateQueries` 刷新列表。

**EmptyState（无公告时）：**
- 图标: lucide-react `Bell` 图标
- 标题: "暂无公告"
- 内容: "点击上方按钮创建第一条公告"

**New 按钮：** 页面右上角，`Button` primary，"新建公告"

**验证：** 组件导出：`export { AdminAnnouncementsPage }`
</action>
