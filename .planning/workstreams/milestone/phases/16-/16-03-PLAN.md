### 16-03-TASK-01: 创建 AnnouncementBanner 横幅组件

<read_first>
`frontend/src/features/upload/UploadPanel.jsx` — Phase 14 更新横幅参考，banner 样式
`frontend/src/shared/ui/index.js` — 可用组件
`frontend/src/features/upload/uploadStatusTheme.js` — 颜色常量参考

<action>
在 `frontend/src/components/` 目录下新建 `AnnouncementBanner.jsx`：

**Banner 样式规范（来自 UI-SPEC.md）：**
- 全宽 banner，固定在页面顶部（下方 nav bar）
- 左边框：4px solid `--upload-brand`（`border-l-4 border-[var(--upload-brand)]`）
- 背景：`bg-secondary`（`oklch(0.97 0 0)`）
- 内边距：12px 16px（`p-3 px-4`）
- 布局：`[关闭 X 按钮] [内容区]`

**Badge：** 标题左侧显示类型 Badge（banner 类型：`bg-upload-brand/10 text-upload-brand` + "公告"）

**验证：** 组件导出：`export function AnnouncementBanner({ announcement, onDismiss })`
</action>

### 16-03-TASK-02: 创建 AnnouncementModal 弹窗组件

<read_first>
`frontend/src/shared/ui/index.js` — Dialog 组件导出
</read_first>

<action>
在 `frontend/src/components/` 目录下新建 `AnnouncementModal.jsx`：

**Modal 样式规范（来自 UI-SPEC.md）：**
- 使用 `Dialog` 组件（shadcn/ui）
- 宽度：`max-w-md`（448px）
- 背景遮罩：半透明深色（Dialog default）
- 边框圆角：`--radius-lg`（shadcn default）

**结构：**
- Header: 标题（18px semibold）+ 右上角 X 关闭按钮（`DialogClose`）
- Body: 内容文字（14px），支持 Markdown 渲染（使用 `react-markdown` 或简单换行处理）
- Footer: "我已知晓" 按钮（`bg-upload-brand` primary，`DialogClose` 包裹，点击关闭）

**Badge：** 标题左侧显示类型 Badge（modal 类型：`bg-destructive/10 text-destructive` + "重要公告"）

**行为：**
- 接收 `announcements` prop（AnnouncementItem[]，可能有多条）
- 维护 `currentIndex` state
- Modal 显示 `announcements[currentIndex]`
- 所有 modal 公告显示完毕后自动关闭
- 用户点击 "我已知晓" 或 X → 显示下一条（或全部关闭）
- 组件内维护 `open` state

**验证：** 组件导出：`export function AnnouncementModal({ announcements })`
</action>

### 16-03-TASK-03: 在 LearningShell 中集成 Banner 和 Modal

<read_first>
`frontend/src/app/LearningShell.js` — 主 layout，页面入口
`frontend/src/shared/api/client.js` — 前端 API 客户端
</read_first>

<action>
在 `LearningShell.js`（或主 App layout）中添加：

1. **导入公告组件：**
   ```jsx
   import { AnnouncementBanner } from "@/components/AnnouncementBanner";
   import { AnnouncementModal } from "@/components/AnnouncementModal";
   ```

2. **获取有效公告：**
   在 `LearningShell` 初始化时（useEffect），调用 `GET /api/announcements/active`
   仅取 `type === "banner"` 的公告传给 Banner
   仅取 `type === "modal"` 的公告传给 Modal

3. **Banner 展示：**
   在页面顶部（nav bar 下方）渲染 Banner 列表
   每条 banner 公告对应一个 `AnnouncementBanner` 实例
   Banner 独立关闭，互不影响

4. **Modal 展示：**
   在 `LearningShell` 根层级渲染 `AnnouncementModal`
   将所有 `type === "modal"` 的公告传入
   Modal 在页面加载时立即显示（自动 open=true）

5. **登录态判断：**
   仅登录用户看到公告（已登录用户才调用 `/api/announcements/active`）

**验证：** 确认 LearningShell 中有 Banner 和 Modal 的渲染逻辑
</action>

<acceptance_criteria>
- LearningShell 导入并渲染 AnnouncementBanner（type=banner 公告）
- LearningShell 导入并渲染 AnnouncementModal（type=modal 公告）
- Banner 在 nav bar 下方独立展示
- Modal 在页面加载时自动弹出
- 未登录用户不调用公告 API
</acceptance_criteria>

**步骤 2：在个人中心页面中内嵌更新日志区域**

在个人中心页面现有内容下方（或作为新 section）添加：

**更新日志 Section：**
- 标题："更新日志"（24px semibold）
- 调用 `GET /api/announcements/active` 仅取 `type === "changelog"` 公告
- 列表按时间倒序（最新在上）
- 每项显示：日期（13px muted）+ 标题（16px semibold）+ Badge（"更新日志"，`bg-secondary`）
- 点击标题展开内容（Markdown 渲染）
- 默认全部折叠，点击后展开

**实现：**
```jsx
const [expandedIds, setExpandedIds] = useState(new Set());

const toggleExpand = (id) => {
  setExpandedIds(prev => {
    const next = new Set(prev);
    if (next.has(id)) next.delete(id);
    else next.add(id);
    return next;
  });
};
```

**EmptyState：** 无 changelog 公告时显示：
- 标题："暂无更新日志"
- 内容："关注 Bottle 最新动态，不错过任何新功能"

**样式参考：** 使用 `ScrollArea` + `Badge` + `Card` 组件

**验证：** 个人中心页面包含更新日志 section
</action>
