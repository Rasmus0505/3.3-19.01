**Direct-upload DashScope ASR now refreshes the signed URL once on FILE_403_FORBIDDEN and returns structured cloud file-access diagnostics when recovery is exhausted**

## Accomplishments

- Added RED/GREEN regression coverage for one-time retry success, retry exhaustion, and task API behavior on direct-upload DashScope 403 failures.

- Attached `dashscope_recovery` metadata to successful lessons and raised dedicated `DASHSCOPE_FILE_ACCESS_FORBIDDEN` errors when retry exhaustion confirms a cloud file-access failure.
