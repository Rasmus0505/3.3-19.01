## Issues Encountered

1. **SQLite "unknown database app" error** — `Base.metadata.create_all()` created tables with `app.` schema prefix. Fixed by setting `schema_translate_map={"app": None}` in engine `execution_options` before `create_all()`.
2. **PUT/POST to deleted endpoints returning 405 instead of 404** — FastAPI matches `/api/admin/subtitle-settings` partial path and returns 405 Method Not Allowed. Updated test assertions accordingly.
3. **`db.get()` bypassing session engine settings** — Direct model-level queries use model's `__table__.schema` regardless of session settings. Worked around by using HTTP endpoint tests instead of direct function tests for billing rates.
