<objective>
Remove the Getting Started guide entirely from the web app. The guide was an onboarding overlay that walked users through upload → history → lesson start. With the desktop-first product direction, this overlay adds no value — users either arrive via desktop guidance or self-navigate via upload/history. Removing it eliminates dead code from LearningShellContainer, LearningShellPanelContent, LearningShellSidebar, and bootstrap.jsx.

Purpose: Simplify the learning shell, eliminate an unused onboarding surface, and make the auth gate uniform across all panels.
</objective>
