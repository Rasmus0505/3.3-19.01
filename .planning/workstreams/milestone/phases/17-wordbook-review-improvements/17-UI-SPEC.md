## 1. Concept & Vision

生词本的核心定位是**复习工具**，收藏是附带能力。界面设计围绕"高频复习、低噪音界面、掌握度反馈"三个目标，让用户在最短路径内完成复习动作，同时提供足够的语境回看来加深记忆。

视觉风格保持现有 shadcn 组件库一致性，复习状态下隐藏非必要信息，专注当前单词。

## 2. Design Language

### Aesthetic Direction
延续现有的 shadcn/ui 风格，复习状态使用深色强调色突出当前词条，列表状态保持简洁卡片式布局。

### Color Palette
- Primary: `hsl(221.2, 83.2%, 53.3%)` (blue-500) — 主要操作按钮
- Secondary: `hsl(210, 40%, 96.1%)` (slate-100) — 卡片背景
- Accent: `hsl(142.1, 76.2%, 36.3%)` (green-600) — 已掌握状态
- Destructive: `hsl(0, 84.2%, 60.2%)` (red-500) — 删除、重来
- Muted: `hsl(215.4, 16.3%, 46.9%)` (slate-500) — 次要文字

### Typography
- 词条文字: 24px, font-semibold
- 语境文字: 14px, text-muted-foreground
- 按钮文字: 14px, font-medium
- 进度条: 12px, text-muted-foreground

### Spatial System
- 卡片圆角: `rounded-2xl` (16px)
- 按钮圆角: `rounded-lg` (8px)
- 内边距: 16px (p-4)
- 间距: 12px (gap-3)

### Motion Philosophy
- 复习结果反馈: 淡入 200ms，显示 1.5s 后淡出
- 进度条: 平滑过渡 300ms ease-out
- 卡片展开/折叠: 200ms ease-in-out

## 4. Features & Interactions

### 4.1 复习进度条
- **位置:** 复习模式顶部，返回按钮右侧
- **显示:** "今天复习进度：X / Y 张" + 进度条 + 百分比
- **计算:** X = 今日已完成复习数，Y = 今日总到期数
- **状态:**
  - 进行中: 蓝色进度条填充
  - 完成: 绿色进度条，显示"今日复习完成 🎉"
  - 无到期: 显示"今天没有到期词条"

### 4.2 复习按钮时间预告
- **位置:** 四个复习按钮下方
- **显示内容:** 每个按钮右侧显示下次复习时间
  - 重来: "10分钟后"
  - 很吃力: "4小时后"
  - 想起来了: "1天后"
  - 很轻松: "4天后"
- **交互:** 用户hover时时间预告高亮
- **计算:** 后端 preview API 返回准确的间隔时间

### 4.3 复习结果反馈
- **触发:** 用户点击任意复习按钮后
- **显示内容:**
  - "✓ 已记录复习结果"
  - "复习间隔：{上次间隔} → {新间隔}（{变化量}）"
- **时长:** 显示 1.5 秒后自动进入下一张
- **动画:** 淡入显示，完成后淡出

### 4.4 语境列表多语境回看
- **默认显示:** 最近 2 条语境
- **展开:** 点击"展开全部"显示所有语境
- **每条语境显示:**
  - 英文句子（可点击发音）
  - 来源课程名称 + 收录时间
  - "查看课程"按钮 → 打开弹窗沉浸学习

### 4.5 弹窗沉浸学习
- **触发:** 点击"查看课程"按钮
- **弹窗大小:** 最大宽度 4xl (896px)，高度 80vh
- **内容:** 简化的沉浸学习播放器（仅播放、字幕、收藏功能）
- **关闭:** 点击 X 或按 ESC，返回复习页面
- **状态保持:** 关闭弹窗后继续复习，不中断复习流

### 4.6 历史复习记录（可折叠）
- **默认:** 折叠状态
- **显示:** 最近一次复习时的语境（英文+中文）
- **用途:** 用户想不起来当前语境时，可以参考历史

## 5. Component Inventory

### ReviewProgressBar
- **Props:** completed: number, total: number
- **States:**
  - Default: 蓝色进度条
  - Complete: 绿色进度条 + 完成文案
  - Empty: "今天没有到期词条"

### ReviewCard
- **Props:** wordbookEntry, onReview, onViewLesson
- **States:**
  - Default: 显示单词、语境、按钮
  - Submitting: 按钮 disabled，显示加载状态
  - Feedback: 显示复习结果反馈（1.5秒后消失）

### ReviewButton
- **Props:** grade, label, intervalPreview, disabled, onClick
- **States:**
  - Default: 白色背景，显示间隔时间
  - Hover: 背景色加深，间隔时间高亮
  - Active/Pressed: 背景色变化
  - Disabled: 灰色，不可点击

### ContextCard
- **Props:** sentence, source, onViewLesson
- **States:**
  - Default: 显示英文句子和来源
  - Expanded: 显示完整句子列表
  - Collapsed: 只显示数量

### ImmersivePopup
- **Props:** lessonId, sentenceIndex, onClose
- **States:**
  - Loading: 显示加载状态
  - Ready: 显示播放器
  - Error: 显示错误信息和重试按钮

## 6. Technical Approach

### Frontend
- **Framework:** React + Vite (现有)
- **UI Library:** shadcn/ui + Tailwind CSS (现有)
- **State Management:** React useState + useCallback (现有模式)
- **New Dependencies:** None required

### Backend

#### 新增 API: 复习时间预告
```
GET /api/wordbook/review-preview/{entry_id}
Response: {
  grades: [
    { grade: "again", interval: "10分钟后", intervalHours: 0.17 },
    { grade: "hard", interval: "4小时后", intervalHours: 4 },
    { grade: "good", interval: "1天后", intervalHours: 24 },
    { grade: "easy", interval: "4天后", intervalHours: 96 }
  ]
}
```

#### 复用现有 API
- `GET /api/wordbook/review-queue` — 获取复习队列
- `POST /api/wordbook/{entry_id}/review` — 提交复习结果（增强返回间隔信息）

### ImmersivePopup 实现策略
1. **不复制沉浸学习代码** — 复用 `ImmersiveLessonPage` 组件
2. **Dialog 包装** — 使用 shadcn Dialog 组件
3. **参数化全屏** — 添加 `isCompact` prop，弹窗模式下禁用全屏功能
4. **隔离播放器逻辑** — 提取 `useSentencePlayback` hook

### Key Implementation Notes
- 复习结果反馈使用 toast + 自定义 duration (1500ms)
- 进度条计算：后端返回 `today_due_count` 和 `today_completed_count`
- 弹窗沉浸学习复用现有课程数据 API
- 历史语境通过 `latest_sentence_en/zh` 和 `source_count` 展示

## 7. Decisions (Locked)

| # | Question | Decision | Rationale |
|---|----------|----------|-----------|
| 1 | "今日"定义 | **A: 自然日** | 更符合用户心理预期，早上复习的卡片不会因为深夜继续计入"今天" |
| 2 | 时间预告精度 | **A: 精确版** | 用户需要准确预期来做决策，模糊语言反而增加认知负担 |
| 3 | 弹窗收藏功能 | **A: 保留** | 复习流程中也可以收藏/取消收藏，功能完整 |
| 4 | 弹窗内跳转 | **B: 允许跳转** | 可以导航，关闭后继续复习当前词条 |
