## Task 1.3: 收紧 openExternalUrl 函数

<read_first>
- desktop-client/electron/main.mjs（第 1041-1048 行的 openExternalUrl 函数）
</read_first>

<action>
将 `main.mjs` 中的 `openExternalUrl` 函数（第 1041-1048 行）替换为白名单版本：

```javascript
async function openExternalUrl(targetUrl = "") {
  const normalizedUrl = trimText(targetUrl);
  if (!normalizedUrl) {
    return false;
  }
  const whitelist = desktopRuntimeConfig?.security?.openExternalWhitelist || [];
  let allowed = false;
  try {
    const parsedUrl = new URL(normalizedUrl);
    allowed = whitelist.some(allowedEntry => {
      try {
        const p = new URL(allowedEntry);
        return parsedUrl.protocol === p.protocol && parsedUrl.host === p.host;
      } catch {
        return false;
      }
    });
  } catch {
    return false;
  }
  if (!allowed) {
    return false;
  }
  await shell.openExternal(normalizedUrl);
  return true;
}
```

确保 `desktopRuntimeConfig` 在 `openExternalUrl` 调用处可访问（`desktopRuntimeConfig` 在模块顶部已声明并由 `initializeDesktopRuntimeConfig` 初始化）。
</action>
