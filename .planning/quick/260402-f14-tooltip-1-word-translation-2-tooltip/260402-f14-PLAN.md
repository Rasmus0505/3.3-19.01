# 260402-F14: 修复生词本翻译保存和Tooltip问题

## 目标
1. 新增 `word_translation` 字段存储单词翻译，避免覆盖原句翻译
2. 修复沉浸式学习 Tooltip 不显示问题

## 字段设计

| 字段 | 用途 | 来源 |
|------|------|------|
| `latest_sentence_en` | 原句（英文语境） | 直接存储 |
| `latest_sentence_zh` | 原句翻译（复用已有翻译） | 无需 API |
| `word_translation` | 单词翻译（需要调用 API） | 调用翻译 API |

## 卡片布局（已确认）

```
┌─────────────────────────┐
│ 📝 apple                │
│ 🔤 苹果                  │  ← 单词翻译（word_translation）
│ ─────────────────────── │
│ 💬 I like apples.       │  ← 原句（latest_sentence_en）
│ 🀄 我喜欢苹果。           │  ← 原句翻译（latest_sentence_zh）
└─────────────────────────┘
```
