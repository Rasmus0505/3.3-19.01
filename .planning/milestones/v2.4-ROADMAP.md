**Status:** ✅ SHIPPED 2026-04-04

**Total Plans:** 8 (8 complete)

## Overview

v2.4 adds CEFR vocabulary intelligence to the Bottle English learning product. Every subtitle word is tagged with its CEFR level (A1-C2/SUPER) at lesson open, cached locally, and displayed as color-coded highlights in the immersive learning UI — green for i+1 (within reach), red for above i+1 (too hard), gray for mastered. Users set their CEFR level (default B1) in their personal center, which persists across devices.

### Phase 25: CEFR 沉浸式展示与历史徽章

**Goal**: 沉浸式学习页面实时展示词汇 CEFR 等级色块（当前句+上一句），历史记录列表标注课程难度，词选入生词本提供流畅动画反馈。

**Depends on**: Phase 24

**Requirements**: CEFR-05, CEFR-06, CEFR-07, CEFR-08, CEFR-09, CEFR-10, CEFR-11, CEFR-16, CEFR-17, CEFR-18

**Plans**:

- [x] 25-01: CSS foundation (CEFR color classes, wordbook animations) + CefrBadge.jsx utility
- [x] 25-02: 本句 CEFR 下划线（答题框词槽）
- [x] 25-03: 上一句 CEFR 色块 + 生词本成功动画（scale + border flash）
- [x] 25-04: 历史记录列表 CEFR 分布徽章 + 背景分析

**Key decisions:**
- SUPER-level words always render as `cefr-above-i-plus-one`, never i+1
- `computeCefrClassName` handles null/undefined/"" as `cefr-mastered` (gray, not red)
- Above-i+1 color: `oklch(0.58 0.24 25)` — distinctly red, not orange
- Wordbook success animation: 200ms scale (1.0→1.08) + 350ms border flash
- `mergeLessonCardMeta` called via `useAppStore.getState()` (Zustand factory pattern)

### Key Accomplishments

1. **CEFR vocabulary infrastructure** — vocabAnalyzer loads 250K-word COCA-derived cefr_vocab.json, analyzes sentences at lesson open, caches results in localStorage with chunked setTimeout(0) execution
2. **Personal center CEFR selector** — A1-C2 RadioGroup with Duolingo-style Chinese descriptions, PATCH API + Zustand dual-write, persists across devices
3. **Immersive CEFR underlines** — Every word slot in the answer box shows a colored underline from lesson start: green (i+1), red (above i+1), transparent/gray (mastered)
4. **Wordbook CEFR color bands + animation** — Previous sentence tokens display CEFR color bands; adding to wordbook triggers scale + green border flash animation
5. **History list CEFR badges** — Lesson cards show CEFR distribution bar + dominant level badge, with background analysis for unanalyzed lessons
