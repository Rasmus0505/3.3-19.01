# Milestone v2.6: 清洗 CEFR 词典数据源

**Status:** ✅ SHIPPED 2026-04-06
**Phases:** 30-31
**Total Plans:** 2

## Overview

将旧 COCA rank-based CEFR 等级替换为权威 CEFR-J Vocabulary Profile 等级，补全词性（POS）信息，修复数据质量问题，为未来 CEFR 等级识别打好基础。

## Phases

### Phase 30: CEFR 词表权威修正

**Goal**: 执行 `fix_cefr_levels.py`，生成包含正确等级、`pos_entries`、版本元数据的修正后词表 `cefr_vocab_fixed.json`
**Depends on**: None
**Plans**: 1 plan

Plans:

- [x] 30-01: 执行 CEFR-J 词表修正 — 运行 fix_cefr_levels.py，生成修正后词表，验证所有 6 项数据质量标准

**Details:**
执行 `fix_cefr_levels.py`，将 7,020 个 CEFR-J 匹配词的等级替换为权威值。生成的 `cefr_vocab_fixed.json` 包含：
- `_vocab_version: "fixed-v1"` 顶级字段

- 未匹配词（约 43,404）保留 rank-based 等级并附加 `_source: "rank-based"` 标记
- 总词数 50,000，无词丢失，无词重复

## Milestone Summary

**Key Decisions:**

- _vocab_version: "fixed-v1" required — 防止静默回退到 SUPER；启用缓存刷新
- pos_entries as array per word — 从最低 POS 派生 primary level；向后兼容
- _source: "rank-based" on unmatched — 区分已验��（CEFR-J）vs 估算（频率）
- Flat 50K-key structure preserved — O(1) lookup via vocabAnalyzer；现有模式验证通过

**Issues Resolved:**

- 5,564 个词等级被修正（84.4% 的 CEFR-J 匹配词）
- 798 个 SUPER 词被提升为有效 CEFR 等级（如 compute: SUPER→B2）
- 所有 14 项 v1 需求全部通过验证

**Issues Deferred:**

- 词形还原/复数合并（ENRICH-01）— 需要额外 NLP 工具，v2+ 再考虑
- 多参考词表合并 — 需要额外数据源，v2+ 再考虑

**Technical Debt Incurred:**

- 无重大技术债务
