    2. Add a `useMemo` near the `currentSentenceTokens` declaration (~line 1281) that:
       - Calls `vocabAnalyzer.analyzeSentence()` with the current sentence text
       - Creates a Map from word (lowercase) → level string
       - Memoizes by `[lesson?.sentences?.[currentSentenceIndex]?.en, cefrLevel]`

       ```javascript
       const currentSentenceCefrMap = useMemo(() => {
         const sentence = lesson?.sentences?.[currentSentenceIndex]?.en;
         if (!sentence || !vocabAnalyzer?.isLoaded) return new Map();
         try {
           const result = vocabAnalyzer.analyzeSentence(sentence);
           const map = new Map();
           for (const tokenInfo of result.tokens) {
             map.set(tokenInfo.word.toLowerCase(), tokenInfo.level);
           }
           return map;
         } catch (_) {
           return new Map();
         }
       }, [lesson?.sentences?.[currentSentenceIndex]?.en, cefrLevel, vocabAnalyzer?.isLoaded]);
       ```

    Note: `vocabAnalyzer` ref is already present in ImmersiveLessonPage.jsx (from Phase 24). If not, add a `useRef` for it.

    Note: For unknown words (not in vocab), `analyzeSentence` returns level "SUPER". This is correct — unknown words always show red/amber CEFR underline.
  </action>
