  <name>Task 2: Configure electron-builder for model packaging</name>
  <files>desktop-client/package.json</files>
  <action>
    Update `desktop-client/package.json` to include the model directory in electron-builder's `extraResources` config so it is packaged into `app.asar.unpacked/`.

    Add to the existing electron-builder configuration:

    ```json
    "electronBuilder": {
      "extraResources": [
        {
          "from": "models",
          "to": "models",
          "filter": ["**/*"]
        }
      ],
      "asar": true,
      "asarUnpack": [
        "models/**/*"
      ]
    }
    ```

    The `asarUnpack` setting ensures files under `models/` are NOT packed into app.asar and are accessible at runtime as `app.asar.unpacked/models/`.

    Per D-10: Model file location is `desktop-client/models/faster-distil-small.en/`. Electron-builder packs this into `resources/app.asar.unpacked/models/`.
