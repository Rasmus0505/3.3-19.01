# Bottle 产品介绍

## 桌面端公开链接工作流

Bottle 桌面端现在提供一条正式的公开链接学习工作流，而不是零散的导入 helper：

`公开链接 -> 本地下载/媒体准备 -> 课程生成 -> 历史/学习`

对外能力承诺固定为：`支持常见公开视频链接`。

支持示例包括：

- `YouTube`
- `B站`
- `常见播客页面`
- `公开视频直链`

这是一条桌面端正式学习工作流，不是通用下载器，也不是受限内容绕过工具。

## 使用边界

- `仅支持公开单条链接`
- `不支持 cookies、账号登录、会员内容、受限内容导入`
- 当素材命中平台限制时，会明确失败并建议改用 `SnapAny`
- 普通用户流程不暴露下载源选择、cookie 管理或底层工具更新入口

## 用户体验原则

- 导入后优先使用解析到的标题作为课程默认标题
- 主流程只稳定利用标题，不把平台、作者、频道、封面做成新的主界面资产
- 成功后默认直达学习
- 如导入失败，用户仍可留在当前表面重试，`SnapAny` 作为 secondary fallback 提供给用户
