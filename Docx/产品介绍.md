# English Sentence Spelling Trainer — 产品介绍

> 产品代号：Bottle，一款以沉浸式跟读练习为核心的英语学习工具。

---

## 一、产品定位

Bottle 解决的是"看了很多英文视频，却没能把听到的内容变成真正记住的句子"这个问题。

传统的背单词应用脱离语境，学了忘、忘了学。我们认为，只有把听到的真实句子完整复述出来，才能真正掌握一门语言的发音、节奏和拼写。Bottle 的核心体验就是：**上传一段英文视频 → 课程自动生成 → 逐句跟读复述 → 积累真实语境中的词汇。**

---

## 二、用户角色

| 角色 | 说明 |
|------|------|
| 学习者 | 上传素材、生成课程、练习跟读、管理生词 |
| 管理员 | 配置系统参数、管理用户、生成兑换码、监控运营数据 |

---

## 三、核心业务流程

### 3.1 生成课程

用户上传一段英文视频或音频，系统自动完成以下工作，最终生成一段可供跟读练习的课程：

```
① 上传素材（mp4 / mp3 / wav / m4a / webm 等格式）
    ↓
② 音频提取
    ↓
③ 自动转写为英文文本（AI 语音识别）
    ↓
④ 英文分句（按语义和标点切分）
    ↓
⑤ 中文翻译（批量）
    ↓
⑥ 生成音频片段（每句话独立的音频）
    ↓
⑦ 课程就绪，可开始练习
```

每一步都在后台自动完成，用户只需上传文件并等待。

课程生成需要消耗点数（来自用户钱包余额）。点数来源为兑换码充值（管理员批量生成）。

### 3.2 拼写练习

进入课程后，学习者以"听 → 理解 → 拼写 → 即时反馈"的循环推进：

```
播放当前句子音频
    ↓
参考中文翻译（播放时被遮挡，暂停时显示）
    ↓
在输入框拼写英文原文
    ↓
实时高亮反馈：逐字标注对/错
    ↓
提交 → 正确则进入下一句
    ↓
重复直至课程完成
```

进度自动保存，中途退出下次继续。

### 3.3 沉浸式练习

在沉浸式练习界面，页面整体变为媒体优先布局，适合专注跟读：

- 大尺寸播放器 + 底部字幕区
- 字幕翻译在播放时遮挡，暂停时松开
- 点击任意单词 → 听该词发音
- 长按单词 → 收藏到生词本（附带上下文英文和中文）
- 选中连续词组 → 收藏为短语

### 3.4 生词本

在学习过程中收集的单词和短语，统一在"生词本"中管理：

- 按来源课程筛选
- 按时间或状态排序
- 标记"已掌握"或删除

生词本是独立于课程的长期积累工具，不随课程删除。

### 3.5 充值与兑换

用户通过兑换码充值点数：

- 管理员在后台批量生成兑换码（设定面额、数量、有效期）
- 用户在"兑换码充值"页面输入码号，余额即时增加
- 余额不足时无法发起新的课程生成

## 五、运营部署

- 部署于 Zeabur，无需自己维护服务器
- 对外提供用户端和管理后台两个入口
- 媒体文件由客户端直传到云端对象存储，不经过应用服务器，节省服务器带宽并保护用户隐私

---

## 六、设计原则

- **真实性**：所有练习内容来自用户自己上传的素材，而非通用语料库
- **即时反馈**：逐字高亮拼写结果，让学生知道具体哪个词记错
- **进度可续**：随时退出，随时继续，不因中断而丢失进度
- **隐私优先**：文件直传云端，不经过应用服务器
- **精准计费**：按实际 ASR 时长和翻译 token 数扣费
